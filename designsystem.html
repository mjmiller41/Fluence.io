LaserKerf.io · Design System

House style, extracted from apps/web/src/styles.css

House style for LaserKerf.io — read this before generating HTML artifacts for this project. Dark, technical, engineering-tool aesthetic; a single orange accent (#f0653a) on near-black panels. Never a light theme.

Fidelity note. This captures the app's actual UI CSS. The styling is real but utilitarian and moderately developed — it's an editor/CAM tool chrome (app bar, toolbar, panels, machine console), not a marketing site. There is one accent color, no declared spacing/type scale variables, and radii/spacing are consistent literals rather than named tokens. Everything below is derived from shipped values; inferred groupings (surface/toolbar/canvas colors, radius scale) are labeled as such.

Foundations — Typography

System font stacks only (no web fonts — offline-first). UI copy uses the sans stack; numeric/console/monospace UI uses the mono stack. Headings tend toward letter-spacing:-0.02em and weight 600.
